Python의 변수는 데이터 타입을 표시할 필요가 없다.
데이터 타입을 알고 싶다면 type 함수 사용.
타입 변환 할때는 str, int, float, bool()을 앞에 씌워주면 됨.

변수의 이름은 숫자로 시작할 수 없으며 주요 함수인 print, input 등은 지양.
변수 명의 사이에는 _를 입력한다 
ex) user_name

무언가를 출력하고 싶다면 print 함수, 입력값을 받고 싶다면 input 함수, 반올림하고 싶다면 round 함수.

파이썬에서 사칙 연산의 우선순위는
1. ()
2. **
3. * or /
4. + or -
이며 기본적으로 왼쪽에서 오른쪽으로 계산이 진행됨.

기본 나눗셈의 경우 float형으로 결과물이 나오나 정수형으로 받기 위해선 // 연산자를 사용하면 됨. 그러나 이는 소숫점을 그대로 자르는 것이므로 주의.
** 연산자는 제곱으로 2 ** 3은 2의 세제곱인 8

만일 string과 int의 데이터를 한번에 출력하고 싶다면 f-String을 사용하면 됨.
ex) print(f"Hello {name}, Welcome to {city}")

Python은 들여쓰기가 중요하며 항상 코드의 가장 마지막 한줄을 공백으로 남겨야 함.
